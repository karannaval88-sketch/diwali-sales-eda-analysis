# 🪔 Diwali Sales Data Analysis (EDA Project)

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on a Diwali Sales dataset to understand customer purchasing behavior, identify popular product categories, and analyze sales trends across different regions.

---

## 🎯 Objective
The main objective is to extract meaningful insights from sales data that can help businesses improve marketing strategies, inventory planning, and customer targeting during festive seasons.

---

## 📊 Dataset
The dataset contains customer and transaction details such as:
- Gender  
- Age Group  
- State  
- Occupation  
- Product Category  
- Purchase Amount  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 🚀 Workflow
1. Data Cleaning and Preprocessing  
2. Handling Missing Values  
3. Exploratory Data Analysis (EDA)  
4. Data Visualization  
5. Insight Generation  

---

## 📈 Key Insights

### 👩 Gender-wise Analysis
- Female customers generated higher total sales compared to male customers  
*(as shown in gender purchase chart)*  

### 🎯 Age Group Analysis
- Age group **26–35 years** contributed the highest purchase amount  

### 🌍 State-wise Sales
- Top performing states:
  - Uttar Pradesh  
  - Maharashtra  
  - Karnataka  

### 💼 Occupation Analysis
- Highest spending seen in:
  - IT Sector  
  - Healthcare  
  - Aviation  

### 🛍️ Product Categories
- Most popular categories:
  - Clothing & Apparel  
  - Food  
  - Electronics  

### 🗺️ Regional Sales
- Central and Southern zones recorded the highest overall sales  

---

## ⚠️ Challenges Faced
- Encoding issues in dataset  
- Missing values  
- Overlapping labels in visualizations  

### ✅ Solutions
- Applied proper file encoding  
- Removed or handled missing data  
- Adjusted chart size and label orientation for clarity

## 📊 Conclusion
This analysis provided clear insights into customer behavior and sales trends during the Diwali festival. The results can help businesses make data-driven decisions in marketing, product planning, and regional targeting.
